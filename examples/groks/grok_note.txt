- Preview grid: `flex: 1; overflow-y: auto` → cuộn tự do theo nội dung
- Upload button: `position: absolute; bottom: 0` → luôn cố định ở dưới cùng col 1
- Thêm `padding-bottom: 90px` trong grid để nội dung không bị che bởi button

### 2. 🗑️ Nút Clear All
- Button `#imgClearBtn` ẩn nếu không có ảnh, hiện khi có ảnh đã chọn
- Hover: icon đỏ với scale animation
- Click: xóa toàn bộ `pendingImages` + toast thông báo

### 4. 📊 Status Cards (like video cards)

| State | Visual | Badge |
|-------|--------|-------|
| `processing` | Shimmer sweep + spinner ring | UPLOADING (pulse animation) |
| `done` | Image thumbnail + zoom overlay | DONE (green) |
| `failed` | Red overlay + error icon | FAILED (red) + error text |

### 5. 📡 `dataImages` — Cấu trúc backend trả về
```json
{
  "<imageGuid>": {
    "imageGuid":      "aaa",
    "uploadGuid":     "xxx",          // nhận dạng batch
    "fileName":       "photo.jpg",
    "status":         "done",         // "processing" | "done" | "failed"
    "isSuccess":      true,
    "linkImage":      "https://...",  // URL ảnh sau khi upload xong
    "fileMetadataId": "uuid",
    "error":          ""
  }
}
```

### 6. ⚡ Smart Grid Update
- Cards được update **in-place** bằng `data-image-guid` attribute
- Chỉ rebuild card nếu **status thay đổi** (processing → done/failed)
- Toast tự động khi ảnh done hoặc failed
